LIVELOOT

LiveLoot is an interactive website that tracks the most popular games in real-time. It gives you a quick overview of both player counts and streaming popularity.

Features

Shows the Top 10 most played games on PC, console, and mobile.

Tracks the Top 10 most watched games on Twitch in real-time.

Clean and visual interface for easily following gaming trends.

Purpose

LiveLoot is perfect for gamers, streamers, and anyone interested in keeping up with the hottest games right now.

Technologies

Real-time statistics from Twitch and other sources.

Responsive design for all devices (PC, consoles, mobile).
